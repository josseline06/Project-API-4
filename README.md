# Duckface - Aplicación Móvil
Aplicación Móvil de la red social DuckFace

### Build and Run

Primero, instalar dependencias usando npm:

```sh
	npm install
	bower install
	gulp
```

Ejecutar la aplicación:

```sh
	Formas:
		* ionic serve --lab
		* ionic run
```

Generar el APK

```sh
	* ionic build <platform>
```
